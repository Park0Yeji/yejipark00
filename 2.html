O-K Text: Hangul Input System in Head Mounted Display Utilizing Gaze and Hand Gestures

    #Virtual_Reality

    #Gaze_Hands_Input

    #Text_Entry_System_Design

[codes] [pdf]

This project is a study on the design of Hangul input system using gaze and hand interactions on an HMD. Considering the consecutive occurrence of consonants and vowels in Korean, the interaction methods are separated. Consonants use gaze tracking, vowels use finger touch, offering a simple and effective character input experience.
